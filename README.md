# @ishiuba.npm.org/styles

Design system and reusable styles for IamSHIUBA projects. Built with Tailwind CSS 4 and SCSS.

## Installation

```bash
npm i @ishiuba.npm.org/styles
```

## Usage

In your main CSS file:

```css
@import "@ishiuba.npm.org/styles";
```

### Customization

You can override the theme variables before importing the library:

```css
:root {
  --accent-color: #ff0000;
}

@import "@ishiuba.npm.org/styles";
```
